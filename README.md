# MESH — Manufacturing Execution System (showcase site)

A static product showcase for **MESH**, a manufacturing execution system for precision
machine shops — live at **https://vishnuce22.github.io/mesh/**

- Plain HTML/CSS/JS. No build step, no framework. Fonts via Google Fonts.
- Screenshots in `assets/screens/` show **demo data only**, captured from a staging
  environment with sensitive fields (part numbers, names, suppliers, branding) blurred.
- Renaming the product: the name appears in `index.html` (title/meta/wordmarks/copy) —
  search for `MESH`; the wordmark split is `MES` (gradient) + `H` (steel).
