# MESH — The Operating System for Modern Manufacturing (showcase site)

Cinematic product showcase for **MESH**, an AI-powered manufacturing execution system
for CNC and precision manufacturing — live at **https://vishnuce22.github.io/mesh/**

- **Stack:** React 18 + TypeScript + Vite + Tailwind CSS 4 + Framer Motion + Three.js (React Three Fiber).
- **Deploy:** GitHub Actions → GitHub Pages (`.github/workflows/deploy.yml`); pushed to `main`, built with base `/mesh/`.
- **Dev:** `npm install && npm run dev`
- Screenshots in `src/assets/screens/` show **demo data only**, captured from a staging
  environment with sensitive fields (part numbers, names, suppliers, branding) blurred.
  All dashboard figures on the page are fictional simulated data.
- Renaming the product: search `MESH` in `index.html` + `src/` (wordmark lives in `src/components/Nav.tsx`).
